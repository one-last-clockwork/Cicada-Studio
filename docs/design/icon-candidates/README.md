# Cicada Studio icon candidates

These are exploratory SVG candidates for the Cicada Studio app icon and favicon.

- `01-signal-shell.svg`: bold cicada silhouette with signal arcs.
- `02-wing-cipher.svg`: geometric cicada with clue/cipher dots.
- `03-night-song.svg`: compact nocturnal mark with sound-wave accents.
- `04-clear-wing-cicada.svg`: more literal cicada with transparent wings, eyes, antennae, and six legs.
- `05-brood-mark.svg`: bold badge-like cicada mark with amber wings and red eyes.
- `06-molting-shell.svg`: cicada-shell inspired mark with a stronger insect outline.

Selected direction: `04-clear-wing-cicada.svg` is applied as `public/favicon.svg`.

Open `preview.html` in a browser to compare app-icon scale and favicon sizes.
